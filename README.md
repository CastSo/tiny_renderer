# Tiny Renderer
A tiny renderer that copies the features of graphics API like OpenGL. Under the guidances of primarily, the ["Tiny Renderer" articles][1].

## CMake build commands 

```
mkdir build
cd ./build
cmake ..
cmake --build ./build
./build/tiny_renderer
```

[1]: https://haqr.eu/tinyrenderer/
